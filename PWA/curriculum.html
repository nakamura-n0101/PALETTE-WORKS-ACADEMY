授業内容

基礎から実践まで、
「つくる力」を身につける。

デザイン・Web制作の基礎から、 実践的なスキルまで。
現場で求められる力を、 段階的に学びます。

施設見学に申し込む 資料請求する

その手で、表現をつくる。

学べる3つのコース

Graphic Design

デザインの基礎から、 印刷物・広告・パッケージなどの ビジュアル制作スキルを学びます。

  • Illustrator / Photoshop
  • デザイン基礎
  • 配色・レイアウト
  • DTP・印刷知識
  • パッケージデザイン

Web Design

Webサイト制作の基礎から、 コーディングやUIデザインまで学びます。

  • HTML / CSS
  • JavaScript基礎
  • レスポンシブデザイン
  • UI / UX基礎
  • Webサイト制作

Portfolio

就職活動に向けた ポートフォリオ制作と キャリアサポートを行います。

  • 作品制作
  • ポートフォリオ制作
  • 履歴書添削
  • 面接対策
  • キャリア相談
